## MaterialsInformatics Course Repository

Portions of this work are derived from the [MaterialsInformatics](https://github.com/sp8rks/MaterialsInformatics) repository by sp8rks, licensed under the MIT License.

**Copyright (c) 2022 sp8rks**

### What We've Incorporated:

- **Jupyter Notebooks**: Educational notebooks from the `worked_examples/` directory covering materials property prediction, featurization, neural networks, Bayesian optimization, and more
- **Course Materials**: Lecture notes and slides covering materials informatics theory and implementation
- **Research Papers**: Curated collection of foundational papers on graph neural networks, transformers, generative models, and materials science applications
- **Code Examples**: Implementations for Materials Project API access, CBFV featurization, crystal graph networks, and various ML models

### License

### Modifications

IMM IT has adapted these materials to focus specifically on:
- Quantum-level structural materials research
- Infrastructure and construction applications
- Smart structural composites and morphogenic materials
- Integration with Project Icarus advanced materials pipeline
